# Afilli Blog — Demo

Basit, statik bir blog demo projesi. Hızlıca göz atmak için `index.html` dosyasını tarayıcıda açabilirsiniz.

Hızlı başlangıç (yerel sunucu önerilir):

```powershell
# Python 3 kullanıyorsan
python -m http.server 8000
# sonra tarayıcıda: http://localhost:8000
```

Dosyalar:
- `index.html` — Ana sayfa
- `styles.css` — Stil ve görünüm
- `script.js` — Küçük etkileşim
- `posts/` — Örnek yazılar

İstersen tema, daha fazla yazı veya bir build sistemi ekleyebilirim.
